# @wshcmx/net types

Этот каталог содержит сгенерированные TypeScript-типы для пакета `wshcmx/net`.

## Содержимое

- `index.d.ts` — файл с объявлениями типов, который генерируется автоматически.
- `package.json` — метаданные пакета `@wshcmx/net`.

## Важно

Файл `index.d.ts` не следует редактировать вручную. В его заголовке явно указано, что он создается проектом `wshcmx/net` при помощи утилиты `Typifier`.

## Как пересобрать типы

Из корня [репозитория](https://github.com/wshcmx/net) выполните:

```bash
dotnet build wshcmx.slnx
dotnet run --project src\Typifier\Typifier.csproj
```

После этого утилита обновит файл `src\Typifier\types\index.d.ts`.

## Назначение пакета

Пакет предназначен для публикации и подключения готовых объявлений типов, соответствующих публичным классам сборки `wshcmx`.
Это позволяет использовать автодополнение и проверки типов в TypeScript-коде, который работает с API библиотеки.
